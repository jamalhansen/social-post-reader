# Social Post Reader

Monitors Bluesky and Mastodon for posts worth *replying to* — not articles to read, but conversations to join. Each morning it surfaces 3–5 reply candidates from voices in your field, with a one-sentence "your angle" generated by a local LLM.

Companion to [content-discovery-agent](../content-discovery-agent), which treats social posts as *signals for articles*. This tool treats the posts themselves as the content.

Part of the [local-first AI tools series](https://jamalhansen.com/series/local-first-ai/).

---

## What it does

1. Fetches text-heavy posts (no link spam) from Bluesky and Mastodon matching your keywords
2. Scores each post for reply value using an LLM and your profile description
3. Generates a one-sentence "your angle" — a thought-starter, not a draft
4. Appends a digest to today's Obsidian daily note under `## Reply Candidates`
5. Optionally stores candidates in SQLite for tracking (did you reply?)

## Installation

```bash
git clone git@github.com:jamalhansen/social-post-reader.git
cd social-post-reader
uv sync
```

## Quick Start

The installed entry point is `social-reader`. After `uv sync`:

```bash
# Fetch and score posts (ollama default)
uv run social-reader run

# Review candidates interactively
uv run social-reader review

# Clear the backlog (mark all 'new' as 'skipped')
uv run social-reader clear

# Show current status summary
uv run social-reader status

# Use Anthropic for scoring
uv run social-reader run --provider anthropic

# Dry run: print candidates without writing to vault
uv run social-reader run --dry-run
```

## CLI Reference

All tools in this series share a common set of CLI flags for model management (`-p`, `-m`, `-n`, `-v`, `-d`) via [local-first-common](https://github.com/jamalhansen/local-first-common).

### Commands

| Command | Description |
|---|---|
| `run` | Fetch posts, score them, and write digest |
| `review` | Interactively mark candidates as replied or skipped |
| `status` | Show summary of tracked candidates |
| `clear` | Mark 'new' candidates as 'skipped' |
